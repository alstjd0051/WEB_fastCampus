color: 글자의 색상

  • rgb(0,0,0) = 검정색
  • 색상 = 기타지정 가능한 색상
    • Google

    text-align : 문자의 정렬 방식

  • left = 왼쪽정렬
  • right = 오른쪽 정렬
  • center = 가운데 정렬
  • justify = 양쪽 정렬

    • text-decoration : 문자의 장식(선)

  • none : 장식없음
  • underline : 밑줄
  • overline : 윗줄
  • line-through : 중앙선
