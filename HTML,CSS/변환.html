요소의 변환 효과

  • transform : 변환함수1변환함수2변환함수3...;
  • transform : 원근법 이동 크기 회전 기울임;

    • 2D 변환 함수

    px

  • translate(x,y) 이동(x축,y축)
  • transalateX(x) 이동 (x축)
  • translateY(y) 이동(y축)

    • 없음(배수)

  • scale(x,y) 크기 (x축,y축)
  • scaleX(x) 크기(x축)
  • scaleY(y) 크기(y축)

    • deg

  • rotate(degree) 회전(각도)
  • skew(x,y) 기울임(x축,y축)
  • skewX(x) 기울임(x축)
  • skewX(y) 기울임(y축)

  • matrix(n,n,n,n,n,n) 2차원 변환 효과

    • 3D 변환 함수

    px

  • translateZ(z) 이동(z축)
  • translate3d(x,y,z) 이동(x,y,z축)
  • scaleZ(z) 크기(z축)
  • scale(x,y,z) 크기(x,y,z축)
  • perspective(n) 원근법(거리)

    • deg

  • rotateX(x) 회전(x축)
  • rotateY(y) 회전(y축)
  • rotateZ(z) 회전(z축)

  • rotate3d(x,y,z,a) 회전 (x,y,z축, 각도)
